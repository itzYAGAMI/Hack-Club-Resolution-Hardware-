# Hack-Club-Resolution-Hardware-
These are my solutions for the challenges for each week for resolution-hardware
## Week-1
For week 1, I built an RC LED Chaser Circuit! This circuit has 3 LEDs which each light up after a 1 second delay after closing the switch and on opening the switch they fade out together!
### The Circuit
![](https://github.com/itzYAGAMI/Hack-Club-Resolution-Hardware-/blob/main/led_chaser.gif)
The circuit has 3 LEDs, this is to simulate a chaser effect. When the switch is closed charge gets stored in the capacitors one by one and lights up the LEDs one by one, I have experimentally made the time interval 1 second. After the switch is opened, all the LEDs collectively fade out because of the stored charge in capacitor.
